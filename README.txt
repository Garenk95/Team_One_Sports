how to start the server:

-> navigate to the /node folder
->make sure you have npm installed
->type "npm install"
->once installed, run the server by typing "npm start"
->if that doesnt work, just type "node server.js"

How to open the homepage

-> type "open ./homepage.html"
->make sure youre in the Team_One_Sports folder
